#Blog Project   
A simple blog project assignment built using Rails and styled with Bootstrap.   

##How to use
* Clone the blog repository
* Start ```rails server```
* View it on localhost:3000   

##Features
* Add, edit, delete posts.   
* Supports simple markdown input.
